Components

Preview of the components used in the app.

📄️ ActivateHomeBanner

Component to activate or manage the home banner display.

📄️ AddAdminModal

Modal dialog for adding a new admin user.

📄️ AddAmenityModal

Modal dialog for adding a new amenity.

📄️ AdminActiveState

Component to display or manage the active state of an admin.

📄️ AdminForm

Form for creating or editing admin users.

📄️ AdminsTableActions

Action buttons for each admin in the admins table. Allows editing, deleting, and managing admin roles.

📄️ AmenitiesTableActions

Action buttons for each amenity in the amenities table. Allows editing and deleting amenities.

📄️ AmenityForm

Form for creating or editing an amenity. Supports validation and submission.

📄️ AuditLogsFilters

A React component for filtering audit logs by target type, action type, and date range. Used in the admin dashboard for filtering logs related to admins and trips.

📄️ BlogPostForm

Form for creating or editing blog posts.

📄️ Breadcrumbs

A navigation component that displays the current page's location within the app hierarchy.

📄️ CommunityCard

Card component for displaying community information.

📄️ CommunityForm

Form for creating or editing a community.

📄️ CommunityTabs

Tabs component for switching between community-related views.

📄️ ConfirmationButton

A button that asks for confirmation before performing an action.

📄️ ConfirmationModal

A modal dialog for confirming user actions, such as deletions or approvals.

📄️ ContentCategorySelect

Dropdown/select for choosing a content category.

📄️ ContentTypeSelect

Dropdown/select for choosing a content type.

📄️ ContentTypeTag

Tag/badge for displaying a content type.

📄️ CourseCategoriesTableActions

Action buttons for course categories in a table.

📄️ CourseCategory

Component for displaying or editing a course category.

📄️ DatePicker

A reusable date picker component, often used in forms for selecting dates.

📄️ DateRangePicker

A date range picker component for selecting a start and end date.

📄️ DeleteAmenity

Button or action to delete an amenity, with confirmation.

📄️ DeleteBlogPost

Button or action to delete a blog post, with confirmation.

📄️ DeleteCourseCategory

Button or action to delete a course category, with confirmation.

📄️ DeleteTrip

A button or action to delete a trip, with confirmation and API integration.

📄️ EditAmenityModal

A modal dialog for editing a trip amenity. Shows a form and handles API submission and errors.

📄️ Editor

A rich text editor component for editing formatted content, used in forms and content management.

📄️ EmailLogs

Component for displaying email logs.

📄️ EmptyState

A component to display when there is no data to show, often with a message and optional action.

📄️ ErrorBoundary

A React error boundary component to catch and display errors in the UI.

📄️ GeoJSONPreview

Component for previewing GeoJSON data on a map or as text.

📄️ HomeBanner

Component for displaying the home page banner.

📄️ HomeBannersTableActions

Action buttons for home banners in a table.

📄️ ImageWithFallback

Image component with fallback support if the image fails to load.

📄️ LanguageSwitch

A component to switch the application's language (e.g., English/Arabic).

📄️ Layout

The main layout component for the dashboard, wrapping all pages and rendering navigation, headers, and content. Handles sidebar, mobile drawer, and user info.

📄️ LayoutToggle

Component to toggle between different layout views.

📄️ LoggedAdminInfo

Displays information about the currently logged-in admin user.

📄️ MakeSuperAdmin

Button or action to promote a user to super admin.

📄️ MobileDrawer

A mobile navigation drawer for the dashboard sidebar. Opens/closes with a button and supports logout.

📄️ NavDropdownItem

A navigation item for dropdown menus in the sidebar. Highlights as active based on the current route.

📄️ NotAuthNav

Navigation component for unauthenticated users.

📄️ NotificationItem

Displays a single notification with title, body, and relative time. Supports marking as read and redirecting on click.

📄️ Notifications

Displays a list or dropdown of notifications for the current user.

📄️ Pagination

Component for paginating lists or tables.

📄️ PostsFilters

Component for filtering blog posts.

📄️ PostsTableActions

A component that renders action buttons for each blog post in a table. Includes view, edit, and delete actions, shown conditionally based on user permissions.

📄️ ProgressBar

A progress/loading bar component, often used as a fallback during page loads.

📄️ ReinstateAdmin

Button or action to reinstate a previously removed admin.

📄️ ReinviteAdmin

Button or action to re-invite an admin user.

📄️ RevokeAdmin

Button or action to revoke admin privileges.

📄️ RichEditor

A rich text editor component for formatted content.

📄️ RouteForm

Form for creating or editing a route.

📄️ RoutesFilters

Component for filtering routes.

📄️ RoutesTableActions

Action buttons for routes in a table.

📄️ SampleComponent

A sample or example component for demonstration or testing.

📄️ SearchInput

A search input component for filtering lists or tables.

📄️ Select

A reusable select/dropdown component.

📄️ SelectUser

Component for selecting a user from a list or search.

📄️ Table

A reusable table component for displaying tabular data.

📄️ TagForm

Form for creating or editing a tag.

📄️ TagsInput

A component for entering and managing a list of tags. Supports adding/removing tags and validation for duplicates/empty values.

📄️ TagsTableActions

Action buttons for tags in a table.

📄️ TermsAndConditionsForm

Form for creating or editing terms and conditions, supporting English and Arabic content and an effective date.

📄️ TermsAndConditionsTableActions

Action buttons for each terms and conditions entry. Allows viewing and editing, with edit disabled if not allowed.

📄️ TitledAvatar

Avatar component with a title or label.

📄️ TitledModal

A modal dialog component with a custom title bar and close button. Used for displaying forms or messages in a modal overlay.

📄️ ApproveTrip

Button component to approve a trip. Triggers a mutation and updates queries on success.

📄️ TripAgenda

Displays a trip's agenda grouped by day, with collapsible sections for each day. Shows activities with start/end times and names.

📄️ TripCard

Displays a summary card for a trip, including title, planner, start date, and action buttons.

📄️ TripStatus

Displays the current status of a trip and provides action buttons for approving, rejecting, or cancelling trips. Shows alerts for each status.

📄️ TripsTableActions

Action buttons for trips in a table.

📄️ UploadDragger

A drag-and-drop file uploader component supporting images and PDFs. Handles file validation, upload, and preview.
